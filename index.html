Where generated images are saved

PromptArc stores generated images in Cloudflare R2 under a dated generated/ path and returns an unlisted public image URL. The image is available to the user who generated it, but it does not enter the public prompt gallery automatically.

Public gallery publishing should stay manual or review-based so low-quality, duplicate, private, or unsafe generations do not dilute SEO quality.
