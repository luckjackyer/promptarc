Course Outline Prompt for ChatGPT

Use this prompt when you need ChatGPT to turn a topic into a structured course outline with modules, lesson objectives, assignments, and a suggested teaching flow.

Prompt template

You are an instructional designer helping build a practical course.

Create a course outline for [topic].

Audience: [who the course is for]
Goal: [what students should achieve]
Course length: [number of lessons or weeks]
Constraints: [platform, experience level, certification needs]

Return:
1. A short course promise
2. Module-by-module outline
3. Learning objectives for each module
4. Suggested assignments or exercises
5. Recommended final project

Keep the curriculum practical, progressive, and easy to teach.

When to use it

This works well for cohort courses, workshops, paid guides, and product onboarding sequences. It is less useful when you need accreditation-level detail or compliance-reviewed curriculum.

How to improve the output

Add the student skill level, format constraints, and one or two example lessons you already know you want. The more concrete the promise, the less generic the outline becomes.
