LinkedIn Post Hook Prompt

Use this prompt to generate stronger post openers that create curiosity without sounding like copied platform fluff.

Prompt template

Write 15 opening hooks for a LinkedIn post.

Topic: [topic]
Audience: [audience]
Point of view: [opinion or lesson]
Goal: [comments, clicks, authority, leads]

Rules:
- Keep each hook under 18 words
- Avoid generic guru language
- Make the hooks specific, human, and credible
- Mix curiosity, contrarian, lesson, and story angles

Return the best 5 first, then the remaining options.

Best use case

This is strongest when you already know the core lesson you want to share. If the idea itself is weak, no hook prompt will fix that.
