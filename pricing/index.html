How credits will work

A standard image generation will use a clear credit amount. Higher-cost actions such as high-resolution output, multiple variations, or batch generation may use more credits, and the exact cost will be shown before the user submits the request.

Failed generations

Failed generations should not become a hidden tax. If the backend fails, the provider returns an error, or no usable image is produced, the credit policy should return or preserve the credits for another attempt.

History and commercial use

Generation history helps users reuse prompts, download results, and repeat useful settings. Users still need to review outputs before commercial publishing, especially for logos, text, people, products, and brand-sensitive scenes.
