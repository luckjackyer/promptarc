# FAQ

## Is this an image generator?

No. This is a prompt pack. You copy the prompts into your preferred AI image tool.

## Can I use the prompts commercially?

Yes, you can use and adapt the prompts in your own workflow. The generated image rights depend on the image tool you use, so check that tool's terms.

## What file format do I get?

You get a Markdown file. It is easy to read, copy, edit, and import into Notion or other tools.

## Which tools does it work with?

The prompts are written to be broadly useful in modern AI image tools such as ChatGPT image tools, Midjourney-style workflows, Ideogram, Leonardo, Firefly, and similar platforms.

## Will this be updated?

Yes. The first launch version contains 50 prompts. Future updates can expand the pack with more categories and examples.
