# Launch Checklist

## Brand and setup

- Pick the final domain and replace `https://www.yourdomain.com`
- Update `config.js` with the real contact email and outbound URLs
- Create a real Gumroad product URL
- Replace default outbound links with approved affiliate links

## GitHub Pages and Cloudflare

- Push the repo to GitHub
- Enable Pages
- Add the custom domain in Pages settings
- Set the DNS records in Cloudflare
- Confirm HTTPS is active
- Confirm apex and `www` resolve correctly

## Search Console and indexing

- Verify the site with a domain property
- Submit `sitemap.xml`
- Request indexing for the homepage, tool page, library hub, and top template pages

## Monetization

- Submit at least one affiliate application
- Add the affiliate disclosure where appropriate
- Publish the first Gumroad pack
- Connect an email endpoint if you want to capture leads

## Content and trust

- Replace placeholder About text with your real point of view
- Review privacy and terms pages for your jurisdiction
- Add one or two real examples from your own usage to the homepage and template pages
