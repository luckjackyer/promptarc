Review queue

Approve only the best user generations.

This admin shell documents the review standard. User submissions stay pending and do not enter the public gallery or sitemap automatically.

Review rubric

  • Quality: clear subject, stable composition, usable output.
  • Uniqueness: not visually repetitive against existing gallery items.
  • Commercial usefulness: helps a real creator, marketer, designer, or operator.
  • Prompt value: prompt teaches a reusable structure.
  • Safety: no trademark misuse, explicit content, or misleading identity use.
